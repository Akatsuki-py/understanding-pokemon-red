# home

ゲームシステム上必要なコードが入っている

home/以下のプログラムはBank0に配置されるためバンクスイッチなどで利用できなくなることはない。

 fade.asm  |  null
 init.asm  |  null
 joypad.asm  |  null
 overworld.asm  |  null
 pic.asm  |  null
 predef.asm  |  null
 serial.asm  |  null

 vblank.asm  |  null
